# Basics

- contains of program units (functions, subroutines, modules)
- a fortran code will contain a main function (which is a program unit itself)
- black spaces are ignored unless they are literal strings
- Data types
  - real
    - 5.5
  - integer
    - 5
  - logical (boolean) 
    - .true.
    - .false.
  - complex
    - (1,3)
  - character (string)
    - 'hello'
